## About Me

I'm a **Computer Science graduate from FAST National University Islamabad Campus** with over **2 years of industry experience** in software development. I'm a passionate **Full-Stack Developer** specializing in building scalable, high-performance web applications using modern technologies like **Next.js**, **React**, and the **MERN stack**.

My expertise extends across multiple domains:
- **Full-Stack Development**: Building end-to-end applications with React, Next.js, Node.js, and various backend frameworks
- **AI/ML Engineering**: Developing machine learning models and AI-powered solutions for real-world applications
- **Data Engineering**: Designing and implementing data pipelines, ETL processes, and data infrastructure solutions

Throughout my career, I've worked with a diverse technology stack including **React**, **Next.js**, **Node.js**, **MongoDB**, **PostgreSQL**, **MySQL**, **Docker**, **Firebase**, **Go**, **Python**, and various AI/ML frameworks. I'm passionate about writing clean, maintainable code and following best practices in software engineering.

I'm an active **open-source contributor** and enjoy collaborating with fellow developers to solve complex problems. I'm always eager to learn new technologies and stay updated with the latest industry trends.
